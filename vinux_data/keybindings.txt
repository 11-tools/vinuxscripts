Vinux Keybindings

This document contains a list of all of the keybindings
for Gnome, Orca, Compiz, Speakup, Yasr  and Bash, as well
as a list of customised Bash Aliases which are available. 
It is divided into nine sections:

1. Gnome Keybindings
2. Compiz Keybindings
3. Orca Desktop Keybindings
4. Orca Laptop Keybindings
5. Bash Aliases
6. Speakup Desktop Keybindings
7. Bash Keybindings
8. Unity Keybindings
9. X-Tile Keybindings

1. Gnome Keybindings

Open the Vinux Manual
    Control + Alt + Q
List keybindings
    Control + Alt + k
Launch web browser
    Control + Alt + w
Home folder
    Control + Alt + n
Root file browser
    Control + Alt + Shift + n
Run terminal
    Control + Alt + t
Launch Gedit text editor
    Control + Alt + g
Restart orca
    Control + Alt + o
Enable 3d desktop
    Control + Alt + 3
Disable 3d desktop
    Control + Alt + 2
Change to default theme/font size 8,12,16,20 or 24
    Windows + Alt + 1,2,3,4 or 5
Change to dark theme/font size 8,12,16,20 or 24
    Windows + Alt + 6,7,8,9 or 0
Toggle monitor on/off
    Control + Alt + m
Enable Vinux Volume Keybindings*
    Windows + Alt + Shift + Up
*Vinux Volume Up
    Windows + Alt + Right
*Vinux Volume Down
    Windows + Alt + Left
*Vinux Toggle Volume Mute
    Windows + Alt + Down
Restart PulseAudio
    Control + Alt + F
